Dashboard Manganês 3D

Histórico do Projeto e Equipe Técnica

Resumo do Projeto

O projeto dashboard_manganes_3d foi desenvolvido para apoiar decisões estratégicas da equipe de geologia da Brasil Minerais, liderada pelo gerente Frederic Vanderoost. A ferramenta consolidou os dados de teores de manganês obtidos por análise XRF, visualizando-os em gráficos interativos com análise metro a metro, média por localidade e mapa georreferenciado.

Programador responsável: Marlon Brito - Analista de segurança do trabalho BMI | Programador FullStack. Suporte técnico: 31982203015

Linha do Tempo de Desenvolvimento

  • 23/02/2025: Solicitação oficial da geologia para apoio da segurança no tratamento de dados XRF (email Frederic BRMin).
  • Projeto LeituraXRFass.py (22 dias): Script desenvolvido para ler PDFs brutos do equipamento X-MET8000, extrair dados por elemento e exportar para Excel estruturado. Base: planilhas como Leitura21012025.xlsx.
  • Estruturação da base de dados ASSAY.xlsx (5 dias): Dados organizados e validados com a equipe técnica.
  • Desenvolvimento do dashboard_manganes_3d (30 dias): Interface com gráficos metro a metro, pizza, histograma, mapa de calor e filtro por cutoff.
  • Implantação e testes finais (5 dias): Validação visual e interativa, integração com GitHub e Render.
